# Lyrne
An open source tutoring platform

# Backend
- Gradle project
- Requires Java 21 (must install)
- Uses the Javalin api framework

#### Run Dev
```
# Linux & MacOS
./gradlew run

# Windows
gradlew.bat run
```

# Frontend
- Node/Npm project (must install)
- Typescript
- Uses the expo app framework

#### Run Dev
```
# Linux & MacOS & Windows
npm install
npx expo start
```
